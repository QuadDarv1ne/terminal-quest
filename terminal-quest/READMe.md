## 📖 О проекте

**Terminal Quest** — автономный HTML-виджет для изучения Linux-команд в игровом формате. Работает на любой платформе без внешних зависимостей.

## 🛠 Технический стек

| Технология | Назначение |
|------------|------------|
| HTML5 | Структура виджета |
| CSS3 | Стили и анимации |
| Vanilla JavaScript | Логика виртуального терминала |
| Web Audio API | Процедурная генерация музыки |
| LocalStorage | Сохранение прогресса игрока |

---

## 📊 Структура уровней

| Блок | Уровни | Тема | Команды |
|------|--------|------|---------|
| 🎯 **Базовый** | 1-5 | Знакомство с терминалом | `pwd`, `ls`, `cd`, `cat` |
| 📁 **Файлы** | 6-10 | Работа с файлами | `ls -a`, `cat`, `touch` |
| 🛠 **Операции** | 11-14 | Скрипты и редактирование | `./script`, `rm`, `mkdir`, `mv` |
| 🌐 **Сеть** | 15-17 | Процессы и сеть | `ping`, `ps`, `kill` |
| 🔐 **Спец.** | 18-24 | Права, SSH, финал | `chmod`, `ssh`, `history` |

**Всего:** 24 уровня

## 💡 Примеры использования

### Изменение темы в терминале

```bash
theme dracula     # Сменить на тему Dracula
theme matrix      # Матричная тема
theme nord        # Тема Nord
```

### Управление музыкой

```bash
music cyber       # Включить Cyberpunk City
music lofi        # Сменить на Lo-Fi
music off         # Выключить звук
```

### Встраивание на сайт

```html
<script src="terminal-quest-v9.js"></script>
<div id="terminal-quest-container"></div>
```

### Через iframe

```html
<iframe 
  src="https://your-site.github.io/terminal-quest/" 
  width="100%" 
  height="700" 
  frameborder="0">
</iframe>
```
